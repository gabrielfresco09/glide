# Glide API

### Pre-requirements

1. Have nvm installed. Otherwise got to https://github.com/nvm-sh/nvm and follow the instructions
2. run `nvm use`. If you don't have the required node version installed, then run `nvm install && nvm use`

### Running the app

1. npm install
2. npm run start

### Runinng tests
1. npm install
2. npm run test
